## Suggested Toolbox Talk Topics

| Topic | Relevant When |
|---|---|
| Working at Height | Scaffolding, roofing, ladders |
| Excavation Safety | Trenching, foundation work |
| Electrical Safety | Wiring, power tools, overhead lines |
| PPE Use & Importance | Daily reminder |
| Manual Handling | Heavy lifting tasks |
| Fire Prevention | Hot work, storage areas |
| Housekeeping | End of shift, site inspection |
| Heat Stress | Summer / hot weather |
| Chemical Handling | Painting, adhesives, solvents |
| Near Miss Reporting | After any near miss event |
| Scaffolding Safety | Before erection or use |
| Confined Space Entry | Tanks, pits, manholes |
